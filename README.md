# Персональный веб-сайт-портфолио

Адаптивный и современный персональный сайт-портфолио, созданный с использованием **React**, **Vite** и **CSS3**. Проект разработан для представления информации о себе, своих услугах, проектах, навыках и контактах.

## 🌐 Демо-версия

[Открыть сайт](https://kairbekov-official.netlify.app)

---

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

### Предпросмотр продакшен-версии

```bash
npm run preview
```

## 🔧 Основные возможности

- Адаптивный интерфейс (мобильные устройства, планшеты, ПК)
- Фиксированное меню навигации с подсветкой активного раздела при прокрутке
- Блок «Обо мне» с описанием навыков и опыта
- Каталог с услугами (разработка сайтов, логотипов, презентаций и др.)
- Раздел с сайтами-проектами (с модальными окнами и ссылками)
- Галерея дизайнов (афиши, логотипы, презентации)
- Контакты с иконками, ссылками и телефонами
- Кастомный курсор (только на десктопе)
- Плавные анимации и переходы
- Lazy loading изображений

---

## 🧰 Используемые технологии

- **React 18** — современная UI библиотека
- **Vite** — быстрый сборщик и dev-сервер
- **CSS3** — адаптивная сетка, кастомные шрифты, медиа-запросы
- **JavaScript (ES6+)** — взаимодействие с DOM, React hooks
- **Netlify** — хостинг и автоматический деплой

---

## 📱 Адаптивность

Сайт полностью адаптивен для:
- 📱 Мобильных устройств (320px+)
- 📱 Планшетов (768px+)
- 💻 Десктопов (1200px+)

---

## 🎨 Особенности дизайна

- Кастомная типографика с использованием Google Fonts (Monoton, Roboto)
- Собственный шрифт Akony для уникальности
- Кастомный курсор с плавным следованием
- Blur-эффекты для секций
- Градиентные разделители между блоками
- Анимации при наведении на карточки и кнопки

---

## 📋 Перед деплоем

1. Убедитесь, что все изображения находятся в `public/img/`
2. Проверьте, что шрифты размещены в `public/font/`
3. Обновите мета-теги в `index.html` при необходимости
4. Запустите `npm run build` и проверьте отсутствие ошибок
